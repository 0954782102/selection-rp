Налаштування для Netlify

Додайте ці змінні середовища у Netlify:

  • SUPABASE_URL
  • SUPABASE_KEY
  • SUPABASE_TABLE_USERS
  • SUPABASE_TABLE_ITEMS
  • SUPABASE_TABLE_DONATIONS (опціонально)
  • PASSWORD_HASH = plain або sha256
  • ROULETTE_COST = 25 (за замовчуванням)

Якщо змінні не встановлені, система працює на локальних тестових даних.
